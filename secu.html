1. Directiva #include

Las directivas #include le indican al compilador que debe importar código de bibliotecas externas antes de compilar tu programa. Esto permite usar funciones ya creadas, como imprimir texto, manejar archivos o trabajar con cadenas.

Ejemplo:


C++

#include // Permite usar cout



 

2. Proceso de compilación

La compilación es el proceso donde el compilador traduce tu código C++ a un archivo ejecutable. Si hay errores de sintaxis, el compilador los detecta aquí.

 

3. Ejecución

Después de compilar, la ejecución es simplemente correr el programa ya traducido. Aquí es donde ves los resultados en pantalla.

 

4. Tipos de datos

Los tipos de datos definen qué clase de información puede almacenar una variable: números enteros, decimales, caracteres, texto, etc. Elegir el tipo correcto hace tu programa más claro y eficiente.

Ejemplos:

C++

int edad = 12; // Número entero
float altura = 1.50; // Número decimal
char inicial = 'A'; // Un solo carácter
string saludo = "Hola"; // Texto (requiere <string>)




5. cout: mostrar información en pantalla

cout es un objeto de C++ que sirve para imprimir texto, números o resultados en la consola. Forma parte de la librería <iostream>.

7. return 0;

Indica que la función main() terminó correctamente. El valor 0 significa “ejecución exitosa”.

 

8. Variables

Son espacios en memoria donde se guarda información que puede cambiar durante la ejecución del programa. Funcionan como “cajas” con nombre.

 

9. Constantes

Son valores que no pueden cambiar durante la ejecución del programa. Se usan para datos fijos como π o límites máximos.

 

12. namespace std

Muchos elementos de C++ (como cout, cin, string, endl) están dentro del namespace std.

Sin usar using namespace std; tendrías que escribir:

C++

std::cout
std::string
std::endl



 

Con la instrucción:

C++

using namespace std; // Permite usar cout, cin, string, endl sin escribir std::



Puedes usarlos sin el prefijo std::.


Código base para todos los programas

C++

#include <iostream>
using namespace std;

// Aquí van las variables

int main()
{
    cout << "Ejemplo de código base en Dev C++";
    return 0;

Tipos de datos

Cadenas de texto (string)

Una cadena de texto almacena una secuencia de caracteres, como palabras o frases. Para usar string necesitas incluir:

C++

#include <string> // Necesario para usar string

string saludo = "Hola"; // Cadena de texto
string nombre = "Alan"; // Almacena palabras o frases
cout << saludo << " " << nombre; // Imprime: Hola Alan



Tipo de dato int

Almacena números enteros, sin decimales. Se usa para edades, contadores, puntuaciones, etc.

C++ 

int edad = 12; // Número entero
int contador = 0; // Variable que puede aumentar o disminuir
int puntos = 150; // Ejemplo de puntuación



 Tipos de datos decimales

Float

  • Ocupa 4 bytes
  • Precisión de 6–7 decimales
  • Es rápido pero menos preciso
  • Útil cuando la precisión no es crítica

Ejemplo:

C++

float pi = 3.141592f;

 

 

Double

  • Ocupa 8 bytes
  • Precisión de 15–16 decimales
  • El tipo decimal más usado
  • Ideal para cálculos moderados

Ejemplo:

C++

double pi_double = 3.14159265358979;

 

 

 

Long Double

  • Ocupa 12–24 bytes (según compilador)
  • Mayor precisión que double
  • Útil para cálculos científicos de alta precisión

Ejemplo:

C++

long double pi_double_long = 3.14159265358979323846L;



Tipo de dato char

char almacena un solo carácter:

  • Letras: 'A', 'Z'
  • Dígitos como caracteres: '5'
  • Símbolos: '#', '?'
  • Espacio: ' '
  • Caracteres especiales: '\n', '\t'

Características:

  • Ocupa 1 byte
  • Guarda el código ASCII del carácter
  • Solo puede almacenar un carácter
  • Si pones más de uno, marca error

Ejemplos:

C++

char letra = 'A';
cout << "La letra es: " << letra << endl;


 

C++

char digito = '7'; // No es el número 7, es el carácter '7'



Tipo de dato bool

bool almacena valores lógicos:

  • true → verdadero
  • false → falso

Características:

  • Ocupa 1 byte
  • Al imprimirse:
    • true → 1
    • false → 0

Ejemplo básico:

C++

bool encendido = true;
bool apagado = false;

cout << encendido << endl; // imprime 1
cout << apagado << endl; // imprime 0



Ejemplo con comparación:

C++

int a = 5;
int b = 10;

bool resultado = a < b; // true
cout << resultado;
