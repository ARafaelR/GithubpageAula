Definiciones

Directivas #include

Las directivas #include le dicen al compilador que debe importar código escrito por otros (bibliotecas) antes de compilar tu programa. Esto te permite usar funciones ya hechas, como imprimir texto en pantalla o manejar archivos.

#include <iostream>  // Permite usar cout

La compilación 

Es el proceso donde el compilador traduce tu código C++ a un archivo ejecutable que la computadora puede entender. Si hay errores de sintaxis, el compilador los detecta aquí.

Ejecución

Después de compilar, la ejecución es simplemente poner en marcha el programa ya traducido. Aquí es donde ves los resultados en pantalla.

Tipos de datos.

Los tipos de datos definen qué clase de información puede almacenar una variable: números enteros, decimales, caracteres, texto, etc. Elegir el tipo correcto ayuda a que el programa sea más eficiente y claro.

 

int edad = 12;          // Número entero

float altura = 1.50;    // Número decimal

char inicial = 'A';     // Un solo carácter

string saludo = "Hola"; // Texto (requiere <string>)


Función cout

Es un objeto de C++ que se utiliza para mostrar información en la pantalla. Forma parte de la librería <iostream> y permite imprimir texto, números o resultados de operaciones hacia la consola.

 

Cadena de texto

Una cadena de texto (o string) es un tipo de dato que almacena una secuencia de caracteres, como palabras, frases o cualquier combinación de letras, números y símbolos.

 

Operadores aritméticos

Los operadores aritméticos son símbolos que permiten realizar operaciones matemáticas en un programa, como 

sumar (+), 

restar (-), 

multiplicar (*), 

dividir (/) o calcular el residuo de una división (%). 

Se usan para manipular valores numéricos dentro del código.

Tipo de dato int

Es un tipo de dato que almacena números enteros, es decir, números sin decimales. Se usa para representar cantidades como edades, puntuaciones o contadores.

return 0;

Es una instrucción que se coloca al final de la función main() para indicar que el programa terminó correctamente. El valor 0 se interpreta como “ejecución exitosa”.

variables

Son espacios en memoria donde se guarda información que puede cambiar durante la ejecución del programa. Funcionan como “cajas” con nombre donde el programador puede almacenar y modificar datos.

Constantes

Son valores que se guardan en memoria, pero no pueden cambiar durante la ejecución del programa. Se usan cuando un dato debe permanecer fijo, como el valor de π o un límite máximo.

Explicaciones extra:

En C++, muchas funciones y objetos útiles, como cout, cin, string, endl, están dentro del namespace llamado std. Sin esta línea, tendrías que escribirlos con su nombre completo, por ejemplo:

  • std : : cout
  • std : : string
  • std : : endl

Cuando escribes using spacename std; , le dices al programa:

"Puedes usar todo lo que esté dentro de std sin poner std: : cada vez."

Declaración de los tipos de datos: 

Float (double, double long) para números con valores decimales.

 

Float

    • Ocupa 4 bytes de memoria.
    • Tiene alrededor de 6 a 7 cifras decimales de precisión.
    • Es el tipo más ligero y rápido, pero también el menos preciso.
    • Se utiliza cuando la precisión no es muy importante.

Sintaxis de ejemplo:

float pi = 3.141592f;

  • ____________________________________________________________________

 Double

    • Ocupa 8 bytes.
    • Tiene alrededor de 15 a 16 cifras decimales de precisión.
    • Es el tipo decimal más usado en C++.
    • Se utiliza cuando se requieren cálculos moderados, como los financieros o científicos. 

Sintaxis de ejemplo:
 

double pi_double = 3.14159265358979;

_____________________________________________________________________

 

Double Long

    • Ocupa 12, 16 o incluso 24 bytes, según el compilador.
    • Tiene más precisión que double, llegando a 18–21 cifras decimales o más.
    • Es el más preciso, pero también el más pesado.
    • se usa cuando la precisión en lo más importante como los cálculos científicos de alta precisión.

Sintaxis de ejemplo:
 long double pi_double_long = 3.14159265358979323846L;

char.

char es un tipo de dato que almacena un solo carácter. Ese carácter puede ser:

    • Una letra ('A', 'Z')
    • Un número como carácter ('5')
    • Un símbolo ('#', '?')
    • Un espacio (' ')
    • Un carácter especial ('\n', '\ t')

Siempre se escribe entre comillas simples ' '.

Puntos importantes para considerar:

    • Ocupa 1 byte de memoria.
    • Internamente almacena un código ASCII, pero tú lo ves como un carácter.
    • Solo puede guardar un carácter a la vez.
    • Si intentas poner más de uno, el compilador marca error.

__________________________________________________________________
 Sintaxis de ejemplo:
 

char letra = 'A';

cout << "La letra es: " << letra << endl;

tipo de dato bool en C++

 

El tipo de dato bool (abreviación de boolean) sirve para almacenar valores lógicos, es decir:

- true → verdadero  

- false → falso  

Es uno de los tipos más simples e importantes en programación, porque permite:

- Tomar decisiones  

- Controlar condiciones   

- Activar o desactivar estados  

- Comparar valores  

 

Características clave

- Ocupa 1 byte de memoria.   

- Solo puede almacenar true o false.  

- Cuando se imprime:

  - true suele mostrarse como 1

   - false suele mostrarse como 0

 

 Ejemplo básico

C++

bool encendido = true;

bool apagado = false;

 

cout << encendido << endl; // imprime 1

cout << apagado << endl;   // imprime 0

 

 

 Ejemplo con comparación

int a = 5;

int b = 10;

bool resultado = a < b;  // true

cout << resultado;   // imprime 1
