practica LogIn

En esta práctica se nos pide crear un formulario de inicio de sesión utilizando HTML y CSS.

El formulario debe incluir campos para el nombre de usuario y la contraseña, así como un botón de inicio de sesión. Además, se requiere que el diseño del formulario sea atractivo y fácil de usar.


HTML


            <div id="ventanaLogIn">
                <div id="opciones">
                    <p>X</p>
                </div>

                <input type="text" id="usuario">
                <input type="password" id="contrasena">

                <button id="ingresar">
                    Login
                </button>
            </div>

JavaScript

                <script>
                    const cuadro = document.getElementById("cuadrado");

                    // Ejemplo 1: Cambiar color cada cierto tiempo (setInterval)

                    // lista de colores
                    let colores = ["dodgerblue", "tomato", "limegreen", "gold", "purple"];
                    let indice = 0;
                    let contador = 0;
                    let velocidad = 1000;
                    let angulo = 0;

                    let intervalo = setInterval(() => {
                        indice = (indice + 1) % colores.length;
                        cuadro.style.backgroundColor = colores[indice];
                        contador += 1;
                        cuadro.textContent = contador.toString();
                        angulo += 90;
                        cuadro.style.transform = \`rotate(\${angulo}deg)\`;
                    }, velocidad);

                    // Detener a los 20 segundos
                    // setTimeout(() => {
                    //     clearInterval(intervalo);
                    // }, 20000);
                </script>
