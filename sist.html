practica LogIn

En esta práctica se nos pide crear un formulario de inicio de sesión utilizando HTML y CSS.

El formulario debe incluir campos para el nombre de usuario y la contraseña, así como un botón de inicio de sesión. Además, se requiere que el diseño del formulario sea atractivo y fácil de usar.


HTML


            <div id="ventanaLogIn">
                <div id="opciones">
                    <p>X</p>
                </div>

                <input type="text" id="usuario">
                <input type="password" id="contrasena">

                <button id="ingresar">
                    Login
                </button>
            </div>
